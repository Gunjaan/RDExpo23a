Code Mania

Collaboration at its finest: Tech enthusiasts unite to showcase their coding prowess in a thrilling tag team event at the annual tech fest.
23rd April 03:00PM - 7:00PM

An event to challenge participants' coding abilities and using time productively. The participants start at one place and code to find there next destination. They are challenged with various coding puzzles including blind coding, error rectification, and some tricky coding problems. Each problem, if solved correctly outputs the next location and the participants need to make their way to the final destination keeping up with time as they have limited time at each step, and if the time runs out, it's GAME OVER!
